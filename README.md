# GDG Eskisehir Events Platform (Phase 1 Foundation)

This repository contains the implementation foundation for a single-organization
community events product:

- Flutter mobile app (user-facing)
- Next.js website (user-facing)
- Next.js admin panel (operations)
- Go backend (shared API/domain rules)

## Current scope

The repository currently includes:

- Phase 1 implementation documents under `docs/phase1/`
- Backend architectural foundation under `backend/`
- Security and secret handling blueprint aligned with Google Cloud Secret Manager

## Non-goals for this commit

- Full production-ready apps are not yet included
- Multi-organization support is intentionally out of scope
- Paid events and waitlist are out of Phase 1 scope
